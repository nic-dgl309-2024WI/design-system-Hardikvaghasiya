    • Flex Containers

    Class Name Uses CSS Property
    .l-flex Displays an element as a block-level flex container display: flex;
    .l-flex-inline Displays an element as an inline-level flex container display: inline-flex;
    .l-flex-row Sets the flex direction of flex items in the container to row flex-direction: row;
    .l-flex-column Sets the flex direction of flex items in the container to column flex-direction: column;
    .l-flex-row-reverse Sets the flex direction of flex items in the container to row reverse flex-direction: row-reverse;
    .l-flex-column-reverse Sets the flex direction of flex items in the container to column reverse flex-direction: column-reverse;
    .l-flex-wrap Allows flex items to wrap onto multiple lines flex-wrap: wrap;
    .l-flex-nowrap Prevents flex items from wrapping onto multiple lines flex-wrap: nowrap;
    .l-flex-wrap-reverse Allows flex items to wrap onto multiple lines in reverse order flex-wrap: wrap-reverse;

    Grid Containers

    Class Name Description CSS Property
    .l-grid Displays an element as a block-level grid container display: grid;
    .l-inline-grid Displays an element as an inline-level grid container display: inline-grid;
    .l-grid-cols-2 Defines a grid with two columns of equal width grid-template-columns: repeat(2, 1fr);
    .l-grid-cols-3 Defines a grid with three columns of equal width grid-template-columns: repeat(3, 1fr);
    .l-grid-rows-2 Defines a grid with two rows of equal height grid-template-rows: repeat(2, 1fr);
    .l-grid-rows-3 Defines a grid with three rows of equal height grid-template-rows: repeat(3, 1fr);
    .l-gap-0 Defines a grid with no gap between the rows and columns gap: 0;
    .l-gap-1 Defines a grid with a small gap between the rows and columns gap: 1rem;
    .l-gap-2 Defines a grid with a medium gap between the rows and columns gap: 2rem;

    Full-width Container Example

    Full-width container adapts to the width of the viewport.

    Max-width container restricts its width on larger screens.
