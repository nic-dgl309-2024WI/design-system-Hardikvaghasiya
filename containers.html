    • Flexbox Utilities

    Use these utility classes to apply flexbox layout properties to elements.

    Flex Container

    .l-flex - Displays an element as a flex container.
    .l-flex-inline - Displays an element as an inline-flex container.

    Flex Direction

    .l-flex-row - Sets the flex direction to row.
    .l-flex-row-reverse - Sets the flex direction to row reverse.
    .l-flex-column - Sets the flex direction to column.
    .l-flex-column-reverse - Sets the flex direction to column reverse.

    Grid Layout Utilities

    Use these utility classes to apply grid layout properties to elements.

    Grid Container

    .l-grid - Displays an element as a grid container.
    .l-inline-grid - Displays an element as an inline-grid container.

    Grid Template Columns & Rows

    .l-grid-cols-2 - Sets the grid to have two columns.
    .l-grid-cols-3 - Sets the grid to have three columns.
    .l-grid-rows-2 - Sets the grid to have two rows.
    .l-grid-rows-3 - Sets the grid to have three rows.
