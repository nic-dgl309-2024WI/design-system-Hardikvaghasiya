    Typography Utility Classes Uses

    Utility Class Description Example
    .u-ff-Playfair Display Applies the Playfair Display font family. Playfair Display Text
    .u-ff-quicksand Applies the Quicksand font family. Quicksand Text
    .u-fw-700 Applies a font weight of 700 (bold). Bold Text
    .u-fw-400 Applies a font weight of 400 (normal). Normal Weight Text
    .u-fs-large Applies a large font size (2.5rem). Large Size Text
    .u-fs-medium Applies a medium font size (1rem). Medium Size Text
