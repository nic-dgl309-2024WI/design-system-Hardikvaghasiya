    Typography Utility Classes Uses

    Utility Class Description Example
    .c-u-ff-Playfair-Display Applies the Playfair Display font family. Playfair Display Text
    .c-u-ff-quicksand Applies the Quicksand font family. Quicksand Text
    .c-fw-700 Applies a font weight of 700 (bold). Bold Text
    .c-fw-400 Applies a font weight of 400 (normal). Normal Weight Text
    .c-fs-large Applies a large font size (2.5rem). Large Size Text
    .c-fs-medium Applies a medium font size (1rem). Medium Size Text
